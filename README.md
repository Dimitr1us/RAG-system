# RAG для решения задач по программированию

**Экспериментальное исследование эффективности RAG (Retrieval-Augmented Generation) при решении задач на Python с помощью Gemini.**

## Описание проекта

Цель проекта — сравнить качество генерации кода большой языковой моделью **с использованием Retrieval-Augmented Generation (RAG)** и **без него**.

Система:
- Сохраняет ранее решённые задачи + их решения в `context.json`
- При поступлении новой задачи ищет наиболее похожие по embedding'ам
- Передаёт похожие примеры в промпт модели
- Сравнивает точность решений с RAG и без RAG
