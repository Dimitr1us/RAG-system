# RAG-system

**Сравнительный анализ Retrieval-Augmented Generation (RAG) и обычной генерации кода с помощью Gemini**

Курсовая работа по дисциплине "Искусственный интеллект"  
**Автор:** Дмитрий Щеглов (ФИ-31-2)

## О проекте

Проект демонстрирует, как **RAG** повышает качество генерации кода языковой моделью Gemini. Система позволяет в реальном времени сравнивать два подхода:
- Генерация **без RAG**
- Генерация **с RAG** (с семантическим поиском похожих задач)

## Основные возможности

- Удобный веб-интерфейс на **Streamlit**
- Семантический поиск по базе задач с помощью эмбеддингов Gemini
- Автоматическое тестирование сгенерированного кода
- Детальный разбор результатов по каждому тесту
- Деплой на Streamlit Cloud
