# RAG-system

**Экспериментальное исследование эффективности RAG при генерации кода на Python с помощью Gemini**

## Описание проекта

Цель курсовой работы — сравнить качество генерации кода большой языковой моделью **с использованием Retrieval-Augmented Generation (RAG)** и **без него**.

### Основные возможности
- Семантический поиск похожих задач через эмбеддинги Gemini
- Генерация решений с RAG и без RAG
- Удобный веб-интерфейс на Streamlit
- Сохранение истории задач и решений в `context.json`
- Деплой на Streamlit Cloud
